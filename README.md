# MIRA AI - Health Prediction Application (v2.5)

An intelligent, multi-module medical dashboard designed to collect patient clinical telemetry, manage active health registry records, and run predictive risk analytics using AI/ML inference logic.

## 🚀 Features
- **Full CRUD Support:** Add, view, edit, and purge patient baseline records.
- **Data Validation Layer:** Validation checking for email structures, chronologically valid dates of birth, and strictly numeric biomarker values.
- **AI/ML Engine Integration:** Dynamic predictive health analysis evaluating risk intervals, displaying insights inside custom clinical telemetry indicators.
- **Telemetry Visualizations:** Interactive real-time metrics charting comparison bars against standard reference target baselines.

## 🛠️ Technology Stack
- **Backend Framework:** Python / Flask
- **Database Layer:** SQLite3 (Persistent Storage)
- **Frontend Layer:** Responsive HTML5 / Bootstrap 5 / Chart.js
